GitHub User-Profile Tool

Description

This tool utilizes the GitHub API to find valid GitHub users based on your search queries. UX theme and functionalities are provided by bootstrap v4.0.0 and jQuery-3.2.1 libraries.
The app executes an AJAX request to the API based on your entries to retrieve the first valid query that match your input. However, for a valid username on GitHub, the app will return the user profile, avatar, repos, gists and other parameters. When there is no match, nothing is displayed.
No frameworks. Just plain Javascript.
